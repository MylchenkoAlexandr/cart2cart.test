# CART2CART

### INSTALLATION
Use `yarn` package manager:  
>`yarn install`

### RUN
Application will be start on `http://localhost:3000/`:  
> `yarn start`  



### BUILD
Work only with NGINX or Apache Server (RTFM React-DOM router)  
>`yarn build`   
> P.S. After building - copy `/static` to `/build` folder!
